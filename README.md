# my_cam
標題：我的自製簡易版桌面錄影<br>
作者：羽山秋人 (http://3wa.tw)<br>
版本：V1.0 (2019-06-23)<br>
<br>
<br>
相關技術：<br>
AviFile 將影片檔與聲音檔結合<br>
Accord 利用裡面的 ffmpeg 來合併截取的圖片<br>
CaptureScreen 包含截取滑鼠位置<br>
NAudio 錄桌面聲音<br>
